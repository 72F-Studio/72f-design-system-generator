# 72F Design System Generator v1.0

![72F Studio](https://img.shields.io/badge/72F%20Studio-Design%20System-blue?style=for-the-badge)
![Penpot v2.14+](https://img.shields.io/badge/Penpot-v2.14+-black?style=for-the-badge&logo=penpot)
![License](https://img.shields.io/badge/License-Apache%202.0-green?style=for-the-badge)

A comprehensive "one-stop shop" Penpot plugin for generating and managing atomic design systems. Built for modern designers who need mathematical precision and accessibility-first foundations in seconds.

## 🚀 Key Features

- **Atomic Design Architect**: Instantly generate Primitives (Spacing, Radii, Shadows) and Semantic Token sets.
- **Native Tokens API**: Fully compatible with Penpot 2.14+ Tokens—no legacy styles or assets.
- **Accessibility Engine**: Real-time WCAG 2.1 AA contrast validation for every semantic color role.
- **Multi-Theme Support**: Effortlessly switch between Light, Dark, and Custom brand modes.
- **High-Fidelity UI**: A premium, ultra-rounded dark interface inspired by HeroUI (NextUI) aesthetics.

## 🛠 Installation

1. Clone this repository.
2. Run `npm install` to install dependencies.
3. Run `npm run build` to generate the production assets.
4. Host the `dist` folder (e.g., using `npx serve -l 4421 --cors dist`).
5. In Penpot, go to **Plugin Manager** > **Install from URL** and enter your hosted `manifest.json` link.
